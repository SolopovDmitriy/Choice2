Скачиваем установщик гит по ссылке
https://github.com/git-for-windows/git/releases/download/v2.25.1.windows.1/Git-2.25.1-64-bit.exe
Принимаем соглашения нажимая next
На следующем экране ищем git bash here и git GUI here, ставим напротив этих двух пунктов галочки
Далее следующие все экраны проклакциваем с next без изменений
Ждем окончания установки

Утилиты Git Bash Here и Git GUI here доступны через контектстное меню (правая кнопка мыши) внутри папки
